connectorPathD : Coordinates -> Coordinates -> String

In the inner `viewProps` function, after each `viewProperty` call, generate a connector:
```elm
viewProps (( x, y ) as coords_) elms =
    case elms of
        [] ->
            ( [], coords_, x )

        element :: elements ->
            let
                ( g_, ( w1, h1 ) ) =
                    viewProperty visited defs collapsedNodes toggleMsg path coords_ element

                connector =
                    connectorPath
                        ( parentRightX, parentY + 14 )
                        ( x, y + 14 )

                ( gs, ( w2, h2 ), w3 ) =
                    viewProps ( x, h1 + 10 ) elements

                maxW =
                    List.foldl Basics.max w1 [ w1, w2, w3 ]
            in
            ( connector :: g_ :: gs, ( x, h2 ), maxW )
```

Each child gets a connector from the parent's right-center to the child's left-center. `parentRightX` is the parent pill's right edge; `x` is the child's left edge (= parentRightX + 10); `y` is the child's top; `y + 14` is the child's vertical center.
