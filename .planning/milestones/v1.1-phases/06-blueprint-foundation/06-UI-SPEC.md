## Color

### Diagram Panel (Dark Blueprint ‚Äî changes in this phase)

| Role | Value | Usage |
|------|-------|-------|
| Dominant ‚Äî diagram background (60%) | `#1a2332` | Full-bleed SVG `<rect>` background fill + `.diagram-panel` CSS fallback |
| Secondary ‚Äî grid dots (30%) | `#2a3a52` | SVG dot grid circles ‚Äî barely lighter than background, subconscious texture |
| Accent ‚Äî node borders (10%) | `#a0c4e8` | SVG pill `stroke` for all regular nodes ‚Äî light blue-white "blueprint ink" |
| Node fill | `transparent` | SVG pill `fill` ‚Äî transparent over dark background (outlined style) |
| Node text | `#e8f0f8` | SVG text fill ‚Äî off-white with cool blue cast |
| Connector lines | `#4a6a8a` | SVG cubic bezier paths ‚Äî muted blue-gray, mid-ground hierarchy |
| $ref node border | `#a0c4e8` dashed | Same hue as regular border, dashed stroke-dasharray retained from Phase 4 |

Source: CONTEXT.md D-01 (background), D-04 (outlined nodes), D-05 ($ref dashed), D-06 (connector ~#4a6a8a).
Node border `#a0c4e8`, node text `#e8f0f8`, grid dot `#2a3a52` are Claude's Discretion values
(CONTEXT.md "Claude's Discretion" section) chosen to satisfy the "light/white borders+text, subtle grid" aesthetic.

Contrast check (against `#1a2332`):
- Node text `#e8f0f8` on `#1a2332`: ~11:1 ‚Äî passes WCAG AA and AAA
- Node border `#a0c4e8` on `#1a2332`: ~5.5:1 ‚Äî passes WCAG AA (non-text border)
- Grid dot `#2a3a52` on `#1a2332`: ~1.2:1 ‚Äî intentionally near-invisible (wallpaper texture, not informational)

## SVG Pattern Contract

The following SVG elements are added at the root `<svg>` level in `Render.Svg.view`. They are not threaded through child view functions.

### Background Rect

```
<rect width="100%" height="100%" fill="#1a2332" />
```

Rendered first ‚Äî behind all diagram content.

### Dot Grid Pattern

```xml
<defs>
  <pattern id="dot-grid" x="0" y="0" width="20" height="20" patternUnits="userSpaceOnUse">
    <circle cx="10" cy="10" r="0.5" fill="#2a3a52" />
  </pattern>
</defs>
<rect width="100%" height="100%" fill="url(#dot-grid)" />
```

Rendered immediately after background rect, before diagram nodes.

Source: CONTEXT.md D-01 (dot radius ~0.5, spacing ~20px), D-02 (SVG `<pattern>` with `<circle>` elements).
